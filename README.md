# @renbostudios/edge-ota CLI

Zero-SDK, self-hostable OTA updates for Expo.

See the [main README](../../README.md) for full documentation.

## Commands

```
edge-ota login          # Log in globally
edge-ota logout         # Log out
edge-ota init           # Set up a new project
edge-ota push           # Publish an OTA update
edge-ota status         # View recent deployments
edge-ota keygen         # Generate a new key pair
```

## Install

```bash
npm install -g @renbostudios/edge-ota
```
